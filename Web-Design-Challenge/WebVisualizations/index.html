Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closee to the equator. To accoplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset over 500 cities, which were picked randomly.

After assembling the dataset, we used Matplotlib to plat various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the anyalsis, as well as explanations and descriptions of any trend and correlations witnessed.
